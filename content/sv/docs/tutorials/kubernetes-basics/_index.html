Kubernetes grunder

Denna handledning ger ett genomgång av grunderna i Kubernetes klusterorkestrationssystem.Varje modul innehåller viss bakgrundsinformation om stora Kubernetes-funktioner och koncept och innehåller en interaktiv onlinehandledning.Dessa interaktiva guider låter dig hantera ett enkelt kluster och dess containeriserade applikationer för dig själv.

Using the interactive tutorials, you can learn to:

  • Distribuera en containeriserad applikation på ett kluster.
  • Skala dina deploys.
  • Uppdatera containeriserad applikation med en ny mjukvaruversion.
  • Debug av containeriserad applikation.

Tutorials använder Katacoda för att köra en virtuell terminal i din webbläsare som kör Minikube, en småskalig lokal distribution av Kubernetes som kan köras var som helst.Det finns inget behov av att installera någon programvara eller konfigurera någonting;Varje interaktiv handledning går direkt från din webbläsare själv.


Vad kan Kubernetes göra för dig?

Med moderna webbtjänster förväntar sig användare att applikationer kommer att vara tillgängliga dygnet runt, och utvecklare förväntar sig att distribuera nya versioner av dessa applikationer flera gånger om dagen.Containerization hjälper till att paketera programvara för att betjäna dessa mål, vilket gör att applikationer kan släppas och uppdateras utan driftstopp.Kubernetes hjälper dig att se till att de containeriserade applikationerna körs var och när du vill och hjälper dem att hitta de resurser och verktyg de behöver för att arbeta.Kubernetes är en produktionsklar, öppen källkodsplattform designad med Googles ackumulerade upplevelse inom containerorkestrering, i kombination med bästa av ras från samhället.
